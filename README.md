# Notes-Api : This is a django based api and a part of full stack notes-app


Developed a cross-platform notes application using Flutter, enabling deployment across Android, iOS, Web, Windows, macOS, and Linux.
Built a Django backend with Django REST Framework (DRF) to handle secure CRUD operations for note creation, editing, deletion, and retrieval.
Implemented RESTful API communication between Flutter frontend and Django backend for seamless data transfer.
Utilized SQLite in the backend for structured data persistence and cloud-based synchronization of user notes.
Enabled real-time syncing to ensure that updates made on one device automatically reflect across all platforms.
Ensured modular, scalable architecture for efficient maintenance and feature extension.
Tech Stack: Flutter | Django | Django REST Framework | SQLite | REST APIs | Cross-Platform Development
